## uniq()

Создает копию оригинального массива, состоящую только из уникальных элементов этого массива.

{% highlight javascript %}
const arrOrg = [1, 2, 3, 1, 2, 3];
const arrUniq = _.uniq(arrOrg);
console.log("uniq array - ", arrUniq); // => [1, 2, 3]
{% endhighlight %}
